Simultaneously Localize, Segment and Rank the Camouflaged Objects

发表于 2024-08-18 分类于 论文笔记 ， Computer Vision ， Salient Object Ranking ， image ， 2021

CVPR 2021

github：JingZhang617/COD-Rank-Localize-and-Segment (github.com)

Abstract

伪装（Camouflage）是许多物种生存的关键防御机制。常见的伪装策略包括背景匹配，即模仿环境的颜色和图案，以及破坏性着色，即隐藏身体轮廓[37]。伪装目标检测 (COD) 的目标是分割隐藏在环境中的伪装物体。现有的 COD 模型基于二元的地面实况，用于分割伪装目标，但未能体现伪装的程度。在本文中，我们重新审视了这一任务，并提出明确地建模伪装物体在特定背景下的显著性，不仅能更好地理解动物的伪装与进化，还能为设计更复杂的伪装技术提供指导。此外，我们观察到伪装物体的某些特定部分使它们对捕食者而言是可检测的。基于对伪装物体的上述理解，我们提出了首个基于排序的 COD 网络 (Rank-Net)，能够同时定位、分割和排序伪装物体。定位模型旨在找到使伪装物体显眼的辨别区域；分割模型用于分割伪装物体的整体范围；排序模型则推断出不同伪装物体的可检测性。此外，我们贡献了一个大型的 COD 测试集，以评估 COD 模型的泛化能力。实验结果表明，我们的模型实现了新的最先进技术，构建了一个更具解释性的 COD 网络。

Domain Separation Graph Neural Networks for Saliency Object Ranking

发表于 2024-08-18 分类于 论文笔记 ， Computer Vision ， Salient Object Ranking ， image ， 2024

CVPR 2024

github：Wu-ZJ/DSGNN (github.com)

Abstract

显著对象排序（SOR）最近引起了广泛关注。以往的方法通常未能明确探索对象之间与显著性程度相关的关系。在本文中，我们提出了一种新颖的域分离图神经网络（DSGNN），该网络首先分别从每个对象中提取形状和纹理线索，并为给定图像中的所有对象构建形状图和纹理图。然后，我们提出了形状-纹理图域分离（STGDS）模块，通过分别明确建模每对对象之间的形状和纹理关系，来分离目标对象的任务相关和无关信息。此外，我们引入了跨图像图域分离（CIGDS）模块，旨在探索对不同场景具有鲁棒性的显著性程度子空间，从而为在不同图像中具有相同显著性水平的目标创建统一的表示。重要的是，我们的 DSGNN 自动学习多维特征来表示每个图边，允许复杂、多样且与排序相关的关系进行建模。实验结果表明，我们的 DSGNN 在 ASSR 和 IRSR 数据集上分别取得了新的最先进性能，SA-SOR 指标有大幅提升，分别为 5.2% 和 4.1%。我们的代码已在 https://github.com/Wu-ZJ/DSGNN 上提供。

Advancing Saliency Ranking with Human Fixations - Dataset, Models and Benchmarks

发表于 2024-08-18 分类于 论文笔记 ， Computer Vision ， Salient Object Ranking ， image ， 2024

CVPR 2024

github：EricDengbowen/QAGNet: Official repository for CVPR 2024 paper "Advancing Saliency Ranking with Human Fixations: Dataset, Models and Benchmarks". (github.com)

Abstract

显著性排序检测（Saliency Ranking Detection, SRD）已经成为计算机视觉中的一个挑战性任务，旨在不仅识别图像中的显著对象，还根据其显著性程度对其进行排序。现有的SRD数据集主要使用鼠标轨迹数据，这不足以捕捉人类视觉感知的复杂性。为了解决这个问题，本文介绍了第一个大规模的SRD数据集SIFR（Saliency Instance Fixation Ranking），该数据集使用真实的人类注视数据构建，从而更贴近真实的视觉感知过程。为了为该数据集建立基准，我们提出了QAGNet，这是一种新颖的模型，利用来自transformer检测器的显著性实例查询特征，在三层嵌套图中进行处理。通过大量实验，我们证明了我们的方法在两个广泛使用的SRD数据集以及我们新提出的数据集上超越了现有的最先进方法。代码和数据集可在https://github.com/EricDengbowen/QAGNet 上获取。

HyperSOR - Context-Aware Graph Hypernetwork for Salient Object Ranking

发表于 2024-08-18 分类于 论文笔记 ， Computer Vision ， Salient Object Ranking ， image ， 2024

TPAMI 2024

MC2文章

github：MinglangQiao/SalSOD: Database for "HyperSOR: Context-aware graph hypernetwork for salient object ranking", TPAMI 2024 (github.com)

Abstract

显著对象排序（Salient Object Ranking，SOR）旨在分割图像中的显著对象，同时预测它们的显著性排序，这些排序是基于人类注意力在不同对象之间的转移而得出的。现有的 SOR 方法主要关注基于对象的注意力，例如对象的语义和外观。然而，我们发现场景上下文在 SOR 中起着至关重要的作用，在不同场景中，同一对象的显著性排序可能会有很大差异。因此，在本文中，我们首次尝试显式地学习场景上下文用于 SOR。具体而言，我们建立了一个包含 24,373 张图像的大规模 SOR 数据集，其中包含丰富的上下文注释，即场景图、分割标注和显著性排序。受我们对数据集分析的启发，我们提出了一种新型的图超网络，名为 HyperSOR，用于考虑上下文的 SOR。在 HyperSOR 中，我们首先开发了一个初始图模块，以分割对象并通过几何和语义信息来构建初始图。接着，设计了一个带有多路径图注意机制的场景图生成模块，用于基于初始图学习对象之间的语义关系。最后，通过一个新颖的图超网络，显著性排序预测模块动态采用所学习到的场景上下文，用于推断显著性排序。实验结果表明，我们的 HyperSOR 可以显著提升 SOR 的性能。

RGB-D salient object ranking based on depth stack and truth stack for complex indoor scenes

发表于 2024-08-17 更新于 2024-08-18 分类于 论文笔记 ， Computer Vision ， Salient Object Ranking ， 3D image ， 2023

PR 2023

Abstract

RGB-D显著性对象检测近年来取得了很大进展，得益于其广泛的应用。以往的研究主要集中在单一对象的简单场景图像上，这些模型通常会在处理复杂的多对象场景时表现不佳。此外，这些方法将显著性对象检测建模为二值分割问题。然而，心理学研究表明，人类会将视觉注意力从一个对象转移到另一个对象，并对显著性对象进行排名，特别是在复杂的室内场景中。基于心理学研究，我们提出在复杂室内场景的RGB-D图像中对显著性对象进行排序。由于缺乏这样的数据，我们首先构建了一个包含多个对象的复杂室内场景的RGB-D显著性对象排名数据集。不同对象的显著性排名基于观察者注意到这些对象的顺序定义。最终的显著性对象排名结果是13位观察者的显著性排名的平均值。该RGB-D显著性对象排名数据集还与当前主流的RGB-D显著性对象检测数据集进行比较分析。由于深度图像提供的位置信息可以帮助确定对象的显著性排名，我们进一步提出了一种端到端网络，利用深度堆叠和真实标签堆叠来预测复杂场景中显著性对象的排序。定量和定性比较结果证明了所提方法的有效性。

Rethinking Object Saliency Ranking - A Novel Whole-Flow Processing Paradigm

发表于 2024-08-17 更新于 2024-08-18 分类于 论文笔记 ， Computer Vision ， Salient Object Ranking ， image ， 2023

TIP 2023

github：MengkeSong/Saliency-Ranking-Paradigm: Official implementation of “Rethinking Object Saliency Ranking: A Novel Whole-flow Processing Paradigm”. (github.com)

Abstract

现有的显著性目标检测方法能够预测突出视觉显著区域的二值图。然而，这些方法在区分多个对象的相对重要性及其相互关系方面存在局限性，这可能导致依赖对象相对重要性的下游任务出现错误并降低准确性。为了解决这一问题，本文提出了一种新的显著性排名范式，旨在通过“重要性顺序”对显著性对象进行全面排名。尽管之前的研究已经展现出一定的潜力，但它们仍然面临一些不适定问题。首先，显著性排名真实值（GT）顺序生成方法不够合理，因为正确确定排名顺序本身并未明确定义，从而导致误报。其次，训练排名模型依然具有挑战性，因为大多数显著性排名方法采用多任务范式，导致不同任务之间存在冲突和权衡。第三，现有基于回归的显著性排名方法对于显著性排名模型来说过于复杂，因为它们依赖于基于实例掩码的显著性排名顺序。这些方法需要大量数据才能准确执行，并且在实施上也具有挑战性。为了解决这些问题，本文对问题产生的原因进行了深入分析，并从“GT数据生成”、“网络结构设计”和“训练流程”三个方面提出了显著性排名任务的全流程处理范式。通过大量实验并在公平合理的比较中，所提出的方法在广泛使用的SALICON数据集上优于现有的最先进方法。显著性排名任务仍处于初级阶段，而我们提出的统一框架可以作为未来研究工作的基本策略。代码和数据将会发布在 https://github.com/MengkeSong/Saliency-Ranking-Paradigm。

Partitioned Saliency Ranking with Dense Pyramid Transformers

发表于 2024-08-17 更新于 2024-08-18 分类于 论文笔记 ， Computer Vision ， Salient Object Ranking ， image ， 2023

ACMMM 2023

github：https://github.com/ssecv/PSR

Abstract

近年来，显著性排名作为一个具有挑战性的任务，专注于评估实例级的显著性程度。由于显著性具有主观性，即使是人类也很难准确识别所有显著实例的确切顺序。以前的方法通常通过直接排序显著实例的排名分数来进行显著性排名，但这些方法并没有明确解决固有的模糊性。为了解决这一限制，我们提出了“分区排名”范式，该方法将无序的显著实例分割成多个分区，然后根据这些分区之间的相关性进行排名。分区排名范式从一般意义上缓解了排名模糊性，并且始终提高了其他显著性排名模型的性能。此外，我们引入了密集金字塔变换器（DPT），以实现全局跨尺度交互，这显著增强了特征交互，并减少了计算负担。大量实验表明，我们的方法优于所有现有方法。我们的方法代码可以在 https://github.com/ssecv/PSR 获得。

Bidirectional object-context prioritization learning for saliency ranking

发表于 2024-08-17 更新于 2024-08-18 分类于 论文笔记 ， Computer Vision ， Salient Object Ranking ， image ， 2022

CVPR 2022

github：GrassBro/OCOR: Bi-directional Object-context Prioritization Learning for Saliency Ranking (github.com)

Abstract

最近提出了显著性排名任务，以研究人类在视觉上通常如何根据不同物体的显著性程度来转移注意力。现有的方法主要关注学习物体-物体或物体-场景关系。这种策略遵循了心理学中的基于物体的注意力概念，但往往倾向于关注具有强语义的物体（例如，人），从而导致不切实际的显著性排名。我们观察到空间注意力与基于物体的注意力在人的视觉识别系统中是同时工作的。在识别过程中，人类的空间注意力机制会在不同的区域（即上下文，context-to-context）之间移动、参与和脱离，这启发我们在显著性排名中除了物体级的推理外，还需要建模区域级的交互。因此，我们提出了一种新的双向方法，将空间注意力与基于物体的注意力统一用于显著性排名。我们的模型包含两个新模块：（1）选择性物体显著性（SOS）模块，通过推断显著物体的语义表示来建模基于物体的注意力；（2）物体-上下文-物体关系（OCOR）模块，通过共同建模显著物体的物体-上下文和上下文-物体交互来分配显著性排名。大量实验表明，我们的方法优于现有的最先进方法。代码和预训练模型可在 https://github.com/GrassBro/OCOR 获取。

Instance-level relative saliency ranking with graph reasoning

发表于 2024-08-17 更新于 2024-08-18 分类于 论文笔记 ， Computer Vision ， Salient Object Ranking ， image ， 2021

TPAMI 2021

Abstract

传统的显著物体检测模型无法区分不同显著物体的重要性。最近，已有两项工作提出通过为不同物体分配不同程度的显著性来检测显著性排序。然而，这些模型中的一个无法区分物体实例，而另一个则更多关注于顺序注意力转移的推断。本文探讨了一种实际问题设置，该设置要求同时对显著实例进行分割并推断其相对显著性排名。我们提出了一种新型的统一模型，作为第一个端到端的解决方案，其中改进的 Mask R-CNN 用于分割显著实例，随后添加了一个显著性排名分支来推断相对显著性。对于相对显著性排名，我们构建了一个新的图推理模块，通过结合四种图来分别融入实例交互关系、局部对比、全局对比和高层语义先验。此外，我们还提出了一种新颖的损失函数来有效训练显著性排名分支。除了这些，我们还为这一任务提出了一个新数据集和评价指标，旨在推动这一领域的研究。最后，实验结果表明，我们提出的模型比以前的方法更有效。我们还展示了其在自适应图像重定向中的实际应用示例。

Salient object ranking with position-preserved attention

发表于 2024-08-17 更新于 2024-08-18 分类于 论文笔记 ， Computer Vision ， Salient Object Ranking ， image ， 2021

ICCV 2021

github: https://github.com/EricFH/SOR

Abstract

实例分割可以检测图像中物体的位置，但难以理解它们之间的关系。我们关注一个典型的关系：相对显著性。一个密切相关的任务是显著物体检测，它预测一个二值图，突出显示视觉上显著的区域，但难以区分多个物体。通过后处理直接将两个任务结合起来也会导致较差的性能。目前对相对显著性的研究不足，限制了内容感知图像裁剪、视频摘要和图像标注等实际应用。在本文中，我们研究了显著物体排序（SOR）任务，该任务旨在根据视觉显著性为每个检测到的物体分配一个排序顺序。我们提出了第一个端到端的 SOR 任务框架（end-to-end framework），并以多任务学习（multi-task learning fashion）的方式解决该问题。该框架同时处理实例分割和显著物体排序。在这个框架中，SOR 分支是独立的，可以灵活地与不同的检测方法配合使用，因此易于用作插件。我们还为 SOR 分支引入了一个专门设计的位置保留注意力（PPA）模块。该模块由位置嵌入阶段和特征交互阶段组成。考虑到位置在显著性比较中的重要性，我们在 ROI 池化操作中保留了物体的绝对坐标，然后在第一阶段将位置信息与语义特征融合。在特征交互阶段，我们应用注意力机制来获得建议框的上下文表示，以预测它们的相对排序顺序。我们在 ASR 数据集上进行了广泛的实验。在没有复杂技巧的情况下，我们提出的方法显著优于之前的最先进方法。代码将公开发布，网址为 https://github.com/EricFH/SOR。
